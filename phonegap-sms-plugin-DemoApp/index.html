SMS Plugin Demo

The SMS plugin allows your app to send text messages on behalf of the user.

On Android you have the choice of sending an SMS by showing the SMS composer UI to the user, or sending the SMS directly to the phonenumber(s) provided. Only the first method is supported on other plaforms.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.
